# servst-cli

  > Simple file server

  [servst][servst] - API for this module

## Install

```sh
npm install --global servst-cli
```

## Usage

```sh
servst --help

  Usage
    servst [dir]

  Examples
    servst
    servst ./static
    servst --port 3100

  Options
    -p, --port <port>  specify port
```
