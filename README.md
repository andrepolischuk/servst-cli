# servst-cli

  > Simple file server

  [servst][servst] - API for this module

## Install

```sh
npm install --global servst-cli
```

## Usage

```sh
servst --help

  Usage: servst [options] [dir]

  Options:

    -h, --help         output usage information
    -V, --version      output the version number
    -p, --port <port>  specify port
```
